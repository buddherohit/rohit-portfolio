# 📦 Installation

Clone the repository

```bash
git clone https://github.com/buddherohit/rohit-portfolio.git
```

Go to project directory

```bash
cd rohit-portfolio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

# 🏗️ Build for Production

```bash
npm run build
```
